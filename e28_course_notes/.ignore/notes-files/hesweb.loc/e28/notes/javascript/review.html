JavaScript review

Prerequisite assumptions

New to JavaScript?

  • Study/reference this guide: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide
  • Bookmark this cheat sheet: Vanilla JavaScript Quick Reference / Cheatsheet

Language architecture:

  • Interpreted
  • Curly-bracket syntax
  • Dynamic typing
  • Object-oriented language based on prototypes rather than classes

Applications:

  • Ubiquitous browser scripting language
    • HTML = Structure, CSS = Style, JS = Interactivity/dynamic behavior
  • Used in other contexts, e.g. on the server w/ Node.js

Browser APIs

JavaScript can work with browser APIs such as:

  • DOM - Document Object Model
    • Treats HTML document as a tree structure
    • Dom methods allow us to access nodes of the tree allowing us to change structure, style, and content of the page
  • XMLHttpRequest to communicate with servers
  • Etc.: Canvas, Audio and Video, Client-side storage

Example

<!doctype html>
<html lang='en'>

<head>
    <title>Example</title>
    <meta charset='utf-8'>
</head>

<body>
    <label for='name'>What is your name?</label>
    <input type='text' name='name' id='name'>
    <button>Say hello</button>

    <script>
        function sayHello() {
            // Extracting data from the DOM
            let name = document.getElementById('name').value;

            let newParagraph = document.createElement('p');
            newParagraph.textContent = 'Hello ' + name;

            // Altering the DOM
            document.body.appendChild(newParagraph);
        }

        const button = document.querySelector('button');
        button.addEventListener('click', sayHello);
    </script>

</body>

</html>
